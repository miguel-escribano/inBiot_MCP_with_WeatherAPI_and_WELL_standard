# Air Quality Parameters Reference

## Environmental Parameters

| Parameter | Unit | Description |
|-----------|------|-------------|
| Temperature | °C | Affects comfort and air circulation |
| Humidity | % | Important for comfort and mold prevention |
| CO₂ | ppm | Indicator of ventilation effectiveness |
| Formaldehyde | µg/m³ | Indoor air pollutant from building materials |
| VOCs | ppb | Volatile Organic Compounds from paints, cleaners |
| CO | ppm | Carbon monoxide from combustion |
| O₃ | ppb | Ozone from outdoor infiltration |
| NO₂ | ppb | Nitrogen dioxide from combustion sources |

## Particulate Matter

| Parameter | Size | Health Impact |
|-----------|------|---------------|
| PM1 | ≤1µm | Ultrafine particles - can enter bloodstream |
| PM2.5 | ≤2.5µm | Fine particles - penetrate deep into lungs |
| PM4 | ≤4µm | Respirable particles - reach lower airways |
| PM10 | ≤10µm | Coarse particles - can reach upper lungs |

## InBiot Composite Indicators (0-100 scale)

| Indicator | Description | Optimal Range |
|-----------|-------------|---------------|
| IAQ | Overall indoor air quality index | ≥80 = Excellent |
| Thermal Indicator | Thermohygrometric comfort | ≥80 = Comfortable |
| Virus Resistance | Conditions affecting pathogen survival | ≥80 = Ideal |
| Ventilation Indicator | Air exchange effectiveness | ≥80 = Excellent |

## Indicator Interpretation

- **80-100**: Excellent conditions
- **60-79**: Good conditions
- **40-59**: Acceptable conditions
- **20-39**: Poor conditions
- **0-19**: Very poor conditions - immediate action required
