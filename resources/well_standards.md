# WELL Building Standard Air Quality Criteria

## WELL Certification Levels

### Platinum (Highest)
- PM2.5: ≤ 8 µg/m³ (annual)
- PM10: ≤ 20 µg/m³ (annual)
- CO₂: ≤ 600 ppm
- VOCs: ≤ 200 ppb
- Formaldehyde: ≤ 9 µg/m³

### Gold Level
- PM2.5: ≤ 15 µg/m³ (24h)
- PM10: ≤ 45 µg/m³ (24h)
- CO₂: ≤ 800 ppm
- VOCs: ≤ 300 ppb

### Silver Level
- PM2.5: ≤ 25 µg/m³
- CO₂: ≤ 1000 ppm (ASHRAE 62.1)
- VOCs: ≤ 500 ppb

| Feature | Parameter(s) | Criteria |

| A03 Ventilation Design | CO₂ | ≤800 ppm operational |
| A05 Enhanced Air Quality | PM2.5, TVOC, HCHO | Strict limits |
| A06 Combustion Control | CO, NO₂ | CO ≤9 ppm, NO₂ ≤40 µg/m³ |

| T06 Adaptive Comfort | Temp, RH | Adjusted to outdoor mean |
| T07 Humidity Control | Relative Humidity | 30-60% |

| Context | Standards |
